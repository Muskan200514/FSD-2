~ Experiment 16: Testing with Pytest

~ Objective
To perform backend testing of a Flask-based Student CRUD API using pyt~ Description
This project implements a RESTful API using Flask and MySQL to manage student data. Pytest is used to test API endpoints such as GET and POST requests.

 ~ Technologies Used
- Python
- Flask
- MySQL
- Pytest

~Test Cases
- Home route test
- Get students API test
- Add student API test

~ Result
All test cases were successfully executed using pytest.

~ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Run tests:
   python -m pytest
