~ Description
- Developed two microservices using Flask:
  - Customer Service → fetch customer orders  
  - Order Service → update order status  
- Data is stored in-memory (no database).

~ Tech Stack
- Python  
- Flask  

~ APIs
- GET `/customer/<id>/orders` → fetch orders  
- PUT `/order/<id>` → update order status  

~ Testing
- Tested using Postman  
- Verified GET and PUT responses 
 
~ Deployment
- Deployed on Render platform
