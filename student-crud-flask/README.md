This project uses MySQL and Python Flask to create a basic student management system. 

It uses RESTful APIs to implement basic CRUD functions, enabling users to add, view, change, and remove student records.

MySQL is utilized for data management and storage, and Flask is used for backend development. 

Postman was used to test API endpoints to make sure they worked properly. 

This project offers a realistic grasp of backend development ideas and shows how to integrate a web framework with a relational database.
